# 🎬 Movie App

A modern movie discovery application built with **React** and powered by **The Movie Database (TMDB) API**. Users can browse trending movies, search for their favorite titles, and view movie details through a clean and responsive interface.

## 🚀 Features

- 🔍 Search for movies by title
- 🎥 Browse trending and popular movies
- 📱 Fully responsive design
- ⚡ Fast and intuitive user interface
- 🎭 View movie information including:
  - Poster
  - Title
  - Rating
  - Release date
  - Overview

---

## 🛠️ Built With

- React
- Vite
- JavaScript
- TMDB API
- CSS / Tailwind CSS

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Logic-Dell/intel-movies.git
```

Navigate to the project folder:

```bash
cd intel-movies/movie-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root and add your TMDB API credentials.

```env
VITE_TMDB_API_KEY=your_tmdb_api_key
```

You can obtain an API key from:
https://developer.themoviedb.org/
